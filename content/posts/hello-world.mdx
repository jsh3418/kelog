## 코드 하이라이팅 테스트

TypeScript 예제:

```typescript
interface Post {
  slug: string;
  title: string;
  date: string;
}

function getLatestPost(posts: Post[]): Post | undefined {
  return posts.sort((a, b) => b.date.localeCompare(a.date))[0];
}
```
